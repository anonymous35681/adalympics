# Adalympics

## Описание исследования
Данное исследование посвящено анализу исторических данных китобойного промысла на основе набора данных [датасета](./data/whaling_data_clean.csv). Основное внимание уделено двум ключевым аспектам:

1. **Динамика продолжительности и успешности рейсов**: Анализ того, как менялось среднее время нахождения судна в море и общая результативность промысла (суммарная добыча ворвани, спермацета и китового уса) с 1730-х по 1920-е годы. Группировка по пятилетиям позволяет выявить долгосрочные тренды, такие как "золотой век" китобойного промысла в середине XIX века.

2. **Сравнительный анализ продуктов промысла**: Исследование относительного вклада различных ресурсов (жир, спермацет, китовый ус) в общую добычу. Нормированные показатели позволяют наглядно сравнить волатильность добычи ценных ресурсов и их корреляцию с увеличивающейся со временем продолжительностью экспедиций.

## Инструкция по запуску
Для генерации графиков и отчета необходимо выполнить следующие команды:
```sh
just run    # постройка 2 графиков
just report # постройка отчёта для ответа на вопросы
```

### Требования
- [Пакетный менеджер uv](https://docs.astral.sh/uv/)
- [Командный раннер just](https://github.com/casey/just)
